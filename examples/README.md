# Example specs

The golden spec corpus — every diagram family ships with at least one sample here. These double as
gallery content and test fixtures.

- **List them all:** `tarseem examples`
- **Render one:** `tarseem render examples/flowchart.json -o out.svg`
- **Which examples cover which family:** [`../docs/guide/families.md`](../docs/guide/families.md)
- **Third-party plugin examples** (a diagram type added without touching core) live in
  [`plugins/`](plugins/) — see [`../docs/extending/clone-a-type.md`](../docs/extending/clone-a-type.md).
